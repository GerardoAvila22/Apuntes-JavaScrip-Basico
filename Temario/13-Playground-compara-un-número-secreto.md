## ****Playground: compara un número secreto****

En este desafío vas a recibir un parámetro `secreto`en la función `solution`. Debes retornar `true` si el número `secreto` es mayor a 5 y `false` si es menor o igual.

Input

```jsx
solution(1)
solution(5)
solution(10)
```

Output

```jsx
false
false
true
```
